# xtrlock-kb
Perl-based wrapper for [xtrlock](https://packages.debian.org/sid/xtrlock).

## Usage

```text
Usage: ./xtrlock-kb.pl options:
  -t=minutes (timeout)
  -b (blank screen after lock)

```
