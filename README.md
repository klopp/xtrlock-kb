# xtrlock-to
Wrapper for [xtrlock](https://packages.debian.org/sid/xtrlock). Lock keyboard and mouse by timeout.

## Usage

```text
Usage: ./xtrlock-to.pl options:
  -t=minutes (timeout)
  -b (blank screen after lock)

```
